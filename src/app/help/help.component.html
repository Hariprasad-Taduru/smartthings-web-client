    • How to setup this tool:

    Version 3.0
    Brief
  • It is zero cost setup tool. Completely instable locally.
  • Makes developer/tester easy working with SmartThings Platform.
  • For best experience, set Chrome Browser to 110% zoom level.
    		•
    In Windows OS
  • Make sure you have installed Java 8.
  • Download executable JAR from MOSOIC page.
  • Under C drive, create folder "st-postman-config"
  • Under folder "st-postman-config", create "env.config" file
  • In file "env.config", add your PAT and location id for each environment as below:

    •         st.stg.pat.token=

            st.stg.test.location=

            st.acpt.pat.token=

            st.acpt.test.location=

            st.prd.pat.token=

            st.prd.test.location=

  • Run below command from command prompt:

    •         java -jar jar-file
    In Ubuntu OS
  • Make sure you have installed Java 8.
  • Download executable JAR from MOSOIC page.
  • Under /etc/ folder, create folder "st-postman-config"
  • Under folder "st-postman-config", create "env.config" file
  • In file "env.config", add your PAT and location id for each environment as mentioned below:

    •         st.stg.pat.token=

            st.stg.test.location=

            st.acpt.pat.token=

            st.acpt.test.location=

            st.prd.pat.token=

            st.prd.test.location=

  • Run below command from bash prompt:

    •         java -jar jar-file




































    More details about this tool:

    Menu Option Description
    Home Page [SmartThings Postman]
    Describes how this tool works. Represented eco system of devices, SmartThings app, Cloud and this web tool.
    Location Tab Displays your favorite configured test location details. You can also list other locations in the seleted environment.
    Devices Tab Displays all your devices under configured test location. You can also fetch particular device full status.
    Rules Tab Displays all automations under configured test location. You can also fetch particular rule definition.
    Scenes Tab Displays all scenes under configured test location. You can also fetch particular scene definition.
    SmartApps Tab Displays all smartapps under configured test location. You can also fetch particular smartapp definition.
    Environment Drop down This tool supports 3 environments (Staging, Acceptance, Production). By default Staging is selected.
    Other details This tool is tested with Google Chrome Browser only. So use Chrome Browser for best experience at 110% zoom.
